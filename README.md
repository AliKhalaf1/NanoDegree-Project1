## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AliKhalaf1/NanoDegree-Project1.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```



<!-- USAGE EXAMPLES -->
## Usage

1. Running server`
   ```js
   npm run start;
   ```
2. Building
   ```js
   npm run build;
   ```
3. testing`
   ```js
   npm run test;
   ```
